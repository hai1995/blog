##### 使用 Python 编写99乘法表

源代码上:  

```
# python 九九乘法表
# 创建外层循环 控制高度
i = 0
while i < 9:
    # 先 += ，从 1 开始 计算
    i += 1
    # 创建内层循环 控制宽度
    j = 0
    while j < i :
        j += 1
        result = i * j
        # 格式化输出内容
        print(f'{j}*{i}={result}',end=" ")
    # 在每行结尾处 进行 换行
    print()
```
